About Perl.com

Since 1997 Perl.com has been the home for quality articles about Perl programming, news and culture. The website is managed by the The Perl Foundation.

The site editor is David Farrell, who can be reached via email. Also on staff is senior editor brian d foy.

Writing for Perl.com

We’re always looking for new authors! Whether you have an event to promote, an experience to share, or a new module or fancy trick to show-off, please see our contributing guide for more information.

Perl

Perl is a general purpose language for getting things done. Created by Larry Wall, the Perl language is managed by The Perl Foundation. For the latest language news and versions, check out Perl.org.
